class DrillPyze.drillstring.BHA(name, filename)[source]

Bases: object

A class describing a bottom hole assembly with the ability to discretize the string

Create a new BHA based on the input excel file. Note, the first component should be a single joint of drillpipe that will go to surface.

Parameters

  • name (str) – Name of the BHA

  • filename (str) – filename of the excel file containing the BHA. Required columns are ‘Component’, ‘Length’, ‘OD’, ‘ID’, and ‘Weight’. ‘Component’ must be one of (‘DP’, ‘HWDP’, ‘DC’, ‘Stab’, ‘Tool’, ‘Bit’) ‘Weight’ should be in lb/ft and is optional

  • bha (DrillPyze.drillstring.BHA) – BHA to be used for the simulation

increment(time, TD_SETPOINT=0, TD_ACT=0)[source]

Increment the simulations with a certain topdrive rpm and recorded top drive torque. The simulations simulate forward at the topdrive rpm and then update their estimation assuming the topdrive torque is measured at the end of the simulation increment.

The simulation pool is then sorted by smallest error and all simulations are cloned to be equal to the best simulation.

Parameters

  • time (float) – Number of seconds to step the simulations forward

  • TD_SETPOINT (float) – Top drive rpm setpoint. Default is 0

  • TD_ACT (float) – Top drive actual torque. Default is 0

DrillPyze.statics.normal_forces(well, bha, WOB=0, TOB=0)[source]

Calculate the normal forces along the drillstring.

Reference: Johancsik, Friesen and Dawson. Torque and Drag in Directional Wells – Prediction and Measurement. Journal of Petroleum Technology. 1984

Parameters

  • well (DrillPyze.well.well_bore) – A well_bore object initialized with a well profile. The wellpath must already be interpolated.

  • bha (DrillPyze.drillstring.BHA) – A BHA object initialized with a drillstring configuration. The BHA must already be discretized.

  • WOB (float) – axial force at the bit (weight on bit). Default value is 0

  • TOB (float) – torque at the bit. Default value = 0
